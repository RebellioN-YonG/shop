# Goshop-backend

Based [vue-element-admin](https://github.com/PanJiaChen/vue-element-admin)

## Project Setup

```sh
npm config set registry https://registry.npmjs.org/

npm install
```

## Compile and Hot-Reload for Development

```sh
npm run dev
```

## Compile and Minify for Production

```sh
npm run build
```

## 提交代码

```shell
# 1. 提交代码前需要安装eslint，进行代码检查
npm config set registry https://registry.npmjs.org/
npm install eslint
# 2. 运行eslint检查代码 
npx eslint --fix src
# 3. 提交代码
git add .
git commit -m "提交说明"
git push origin master
```
